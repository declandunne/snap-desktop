Graticule Overlay (Map Gridlines)

   Graticule Overlay (Map Gridlines)

Toggles the overlay of the graticule layer within a band's Image View. The overlay style properties can be modified using the Graticule overlay page in the Preferences Dialog.

The Graticule Overlay (Map Gridlines) tool creates a gridlines layer over the image which can include coordinate labels, gridlines and a border.

derived from a MODIS Aqua level-2 scene

Location

Layer Toggle - The graticule layer is toggled on and off via the graticule icon in the Overlay toolbar or Layer->Graticule Overlay within the menu system. Initially the graticule layer does not exist, but is created during the first instance of the graticule layer being toggled on. Default settings are applied when the graticule layer is created.

Layer Editor - Subsequent to the creation of the graticule layer, any of the properties such as font size, line spacing, etc., may be edited by using the Layer Editor. This editor is accessed by opening the Layer Manager, clicking on the word Graticule (this selects the layer as the active layer), and then either clicking of the pencil icon within the Layer Manager or selecting Layer->Layer Editor from within the menu system.

Preferences Editor - To edit the default values of the graticule layer properties go to Tools->Options->Layer->Graticule Layer.


Properties

Grid Spacing - The latitude and longitude step sizes may be set directly where the value represents degrees between the gridlines. Setting this field to zero results in Auto-Sizing. The Auto-Sizing algorithm will set the gridline spacing to a reasonable value such that there are about 3-5 lines. Also note that lines are placed at the nearest most sensible degree value.

  • Latitude - sets latitude step size
  • Longitude - sets longitude step size

Labels - Formatting and placement of the text labels associated with the map gridlines indicating geographic position.

  • North - Display north labels
  • South - Display south labels
  • West - Display west labels
  • East - Display east labels
  • Suffix (N,S,W,E) - Format label text with suffix (N,S,W,E) instead of (+/-)
  • Decimal Value - Format label text with decimal value instead of degrees/minutes/seconds
  • Put on Inside - Put labels inside of the data image. In this case a backdrop is added to the labels.
  • Italic - Format label text font in italic
  • Bold - Format label text font in bold
  • Rotation (Latitude) - Rotate latitude labels (0 degrees = perpendicular)
  • Rotation (Longitude) - Rotate longitude labels (0 degrees = perpendicular)
  • Font - Set the font of the labels
  • Size - Set the size of the labels
  • Color - Set the color of the labels

Gridlines - Formatting of the map gridlines.

  • Show - Display the gridlines
  • Width - Set the width of the gridlines
  • Dash Length - Set dash length of gridlines or set to solid gridlines (0=SOLID)
  • Transparency - Set the transparency of the gridlines
  • Color - Set the color of the gridlines

Border - Formatting of the image border.

  • Show - Display the border
  • Width - Set the width of the border
  • Color - Set the color of the border

Tickmarks - Formatting of the tickmarks.

  • Show - Display the tickmarks
  • Put Inside - Placement of the tickmarks towards the inside of the image
  • Width - Set the width of the tickmarks
  • Color - Set the color of the tickmarks

Corner Labels - Display the labels for the corner pixels of the image.

  • North - Display the north corner label
  • South - Display the south corner label
  • West - Display the west corner label
  • East - Display the east corner label

Inside Labels Backdrop - Formatting of the backdrop used when displaying labels inside of the image.

  • Transparency - Set the transparency of the inside label backdrop
  • Color - Set the color of the inside label backdrop

Notes, Pitfalls and Issues

Pitfall: Blurry Text Labels - Scene images are exported using a raster file format (png, gif, etc.). This means that any text overlayed on the image such as the map gridlines labels and other text annotations will also be rasterized. In order to maintain the appearance of sharp text labels you may need to increase the image dimension of the file to be exported. This can be done within the Export Image tool.

Pitfall: No Labels because they are Outside of View Window - When labels are placed outside of the image border this will also likely place them outside of the view window, in which case so you will need to zoom out to see them. When you export the image using the Export Image tool, you will need to specify "View region" and not "Full scene" as the Image Region.

Font Sizing - The user font size property is in term of "approximated" points rather than pixels. This enables a sensible default font size setting which will work well with files of any resolution. Internally the font sizes are actually tied to the pixel sizes and there is a conversion of points to pixels being made.

Small Image Issues - For extremely small files (less than about 100 pixels in one dimension) you may likely see problems with the map gridlines, such as missing labels.
